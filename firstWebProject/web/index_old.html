클라이언트 페이지로 부터 서버로 서비스를 요청하는 방법

서비스 요청시 데이터 전송도 포함됨

1. form 태그 사용 : action = "요청받을 서버쪽 대상의 이름"

=> 해당 폼 안에 입력된 데이터들이 submit 버튼 누르면 대상에게로 전송됨.

2. a 태그 사용 : href = "요청 처리할 서버쪽 대상의 이름"

=> a 태그에서도 데이터 전송이 가능함
=> href = "대상이름?변수명=값&변수명=값"
=> ?변수명=값&변수명=값 : 이 부분을 쿼리스트링 이라고 함

3. ajax 이용하는 방법 : 자바스크립트 코드에서 사용함
